ctrl + / : 해당 언어별 주석처리
ctrl + 좌/우 : 단어선택
ctrl + home/end : 해당 페이지 제일 상단/하단 이동
ctrl + b : 왼쪽 패널 열기/닫기
ctrl + d : 같은 단어 선택
ctrl + p : 파일 검색
ctrl + t : 심볼 검색 (특정 심볼이름 검색, ex) 함수명, 클래스명 등)
ctrl + shift + o : 페이지 내에서 심볼 전부 보기
ctrl + shift + c : 외장 터미널 열기
ctrl + shift + \ : 블록 끝으로 이동
ctrl + shift + . -> 편집 상단 내비게이션 활성화 -> ctrl + 좌/우

alt + 위/아래 : 해당 줄 위/아래로 이동
alt + click : 클릭한 곳마다 커서 만들어준다.
alt + shift + click : 세로줄 커서 만들어준다.
alt + shift + i : 세로줄 끝에 커서 만들어준다.
alt + shift + 위/아래 : 해당 줄 위/아래로 복사
alt + shift + 좌/우 : 단어기준 좌우로 확장하면서 선택

alt + 1, 2, 3 ... : 편집창 왼쪽 순서대로 1, 2, 3 ...
ctrl + ` : 하단 터미널 열기 혹은 이동
ctrl + 0 : 왼쪽 패널 이동
ctrl + 1 : 편집 영역으로 이동

해당 단어에 커서, 혹은 해당 단어 선택 (다른 페이지 전부 변경된다.)
f2 : 정의된 이름, 사용된 부분 일괄 변경
f12 : 정의된 이름, 사용된 부분 순차적 이동
alt + shift + f12 : 왼쪽 패널 search 영역에 정의된 이름, 사용된 페이지 전부 찾아서 보여준다.
